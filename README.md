# GroceryMart-Coffee
Start your business journey by opening a small grocery store.

<img src="Thumbnail.png" alt="Project Thumbnail" width="100%"/>
**GroceryMart-Coffee** is an eCommerce platform focused on offering a wide range of grocery products, specifically high-quality coffee. With a modern design, user-friendly interface, and smooth user experience, GroceryMart-Coffee allows customers to easily browse, select, and purchase coffee products from the comfort of their homes.

## Key Features

- **Extensive Product Categories**: From coffee beans, ground coffee, to premium products from well-known brands like Lavazza.
- **User-friendly Interface**: Customers can effortlessly browse products by price, coffee type, or brand with a simple and intuitive interface.
- **Online Ordering Support**: Easily add products to the cart and complete the purchase in just a few clicks.
- **LocalStorage Integration**: Page content is stored locally, allowing faster browsing and optimized resource usage.
- **Real-time Product Updates**: Product data is fetched and updated from the server, ensuring that users always see the latest information.

## Project Goals

The aim of this project is to build a professional eCommerce platform tailored to coffee and grocery stores, where customers can easily find and buy their favorite products. The project also strives to deliver a simple, convenient, and optimized shopping experience.

## Installation Guide

1. Clone the repository from GitHub:
   ```bash
   git clone https://github.com/Vo-Dinh-Quan/GroceryMart-Coffee.git
   ```

2. Open the application:
   - Open the `index.html` file in your browser to start exploring the application.

## Technologies Used

- **HTML5/CSS3**: Used for building the user interface.
- **JavaScript**: Handles logic, including interacting with localStorage and fetching data from the server.
- **Responsive Design**: Ensures that the website looks good on all devices, from desktops to mobile phones.
